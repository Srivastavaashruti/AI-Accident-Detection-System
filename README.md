# AI-Accident-Detection-System
An AI-powered web application developed using Python, Flask, and Machine Learning to detect road accidents from uploaded images/videos and provide emergency alert features.

The system analyzes accident severity, stores accident records in a database, and supports live location tracking with Google Maps integration.

**Features**

*AI-based accident detection
*Accident severity prediction
*Live location tracking
*Google Maps integration
*Dashboard for accident records
*Login authentication system
*SMS and Email alert support
*Real-time web interface using Flask

**Technologies Used**

-Python
-Flask
-Machine Learning
-Flask-SocketIO
-SQLite Database
-HTML/CSS
-JavaScript
-Geopy
